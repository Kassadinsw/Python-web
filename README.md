# 北京化工大学 Python 数据处理课程设计

------

Python数据处理最终作业，包含文档，PPT和个人感想。

## 项目简介

> * 使用Python 爬虫爬取电商网站京东，亚马逊，苏宁的商品信息，并将商品信息返回给前端界面。
> * 使用Python Django框架来进行web界面的开发和数据库的交互。
> * 通过JavaScript来进行网页炫酷效果的编写。
